## 单变量时间按预测

data：存放数据文件

saved_models：存放训练的模型

config_1.json：配置文件

训练并且预测数据：

```Linux
python train.py
```

用训练好的模型预测数据：

```Linux
python pre.py
```

## 多变量时间序列预测

用线性回归实现：

python mult-linear.py
